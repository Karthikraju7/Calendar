# 📅 Interactive Calendar Component

A **modern, responsive, and interactive calendar UI** built using **React (CRA) + Tailwind CSS**.
Designed to mimic a **wall calendar experience** with clean UI, smooth interactions, and practical productivity features.

---

## ✨ Features

### 🎨 Seasonal Theme System

* Each month has a **unique color theme**
* Inspired by **seasons (primary)** with subtle aesthetic variations
  
---

### 📆 Date Range Selection

* Select **start and end dates**
* Highlights:

  * Start date
  * End date
  * Range in between
* Controlled interaction using a **"Select" confirmation button**

---

### 📝 Notes + Todo System

* Add **notes and tasks** to:

  * A single date
  * A selected date range
  * All weekdays (e.g., Mondays, Sundays)

#### Modal Layout:

* **Left:** Notes (textarea)
* **Right:** Todo list with:

  * Add tasks
  * Mark complete/incomplete
  * Filter (All / Unchecked)

---

### 📌 Month Notes Panel

* Dedicated notes section for each month
* Includes a **motivational quote**
* Persistent using **localStorage**

---

### 🔴 Smart Indicators

* Dates with notes/todos show a **dot indicator**
* Automatically removed when data is cleared

---

### 📱 Fully Responsive

* **Desktop:** All content fits in one screen
* **Mobile:** Stacked layout with smooth usability

---

## 🛠 Tech Stack

* **React (Create React App)**
* **Tailwind CSS**
* **localStorage** (for persistence)

---

## 🎯 Design Highlights

* Wall calendar inspired layout
* Hero image per month
* Clean color system per month
* Compact and readable calendar grid

---
